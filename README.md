Trabalho de Grau B - Processamento Gráfico 2023/02

Desenvolvimento de um simples aplicativo com edição de efeitos de imagem e vídeo

# Quickstart

1. Instale (se não tiver instalado) o módulo Pillow

```
pip install Pillow
```

2. Execute o seguinte comando na raíz do projeto:

```
python prototipo.py
```
